# Changelog

All notable changes to the Consider extension are documented in this file.

## 0.1.0

- Initial public release of Consider extension capabilities.
- Inline comment threads using VS Code Comments API.
- Shared `.consider/store.json` model with atomic writes.
- Agent integration setup flow with project/home skill install options.
- Content-based re-anchoring with stale/orphan detection.
- Thread workflow and anchor state split model.
- Consider comments tree view, archive resolved workflow, and reconcile command.
- Extension Host integration tests and UI smoke test harness.
